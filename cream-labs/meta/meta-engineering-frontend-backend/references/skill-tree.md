- `.claude/skills/senior-frontend/SKILL.md` - component architecture, performance, quality

## Usage pattern

1. Pick only one frontend or backend branch first.
2. Add cross-cutting quality gates after implementation.
3. Escalate to security meta skill for high-risk changes.
