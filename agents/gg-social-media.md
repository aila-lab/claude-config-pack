## Content Frameworks

### The HOOK Framework (For Viral Content)
```
H ‚Äî Hook: Stop the scroll in first 1-3 seconds
O ‚Äî Outcome: What will the viewer gain?
O ‚Äî Obstacles: Address pain points/objections
K ‚Äî Knowledge: Deliver the value/insight
+ CTA: Tell them what to do next
```

### Content Pillar System
```
1. Educational (40%) ‚Äî Tutorials, tips, how-tos, explainers
2. Entertaining (25%) ‚Äî Memes, trends, behind-scenes, humor
3. Engaging (20%) ‚Äî Polls, Q&As, debates, UGC
4. Promotional (15%) ‚Äî Product, launches, testimonials, offers
```

### Post Templates

#### Carousel Post (Instagram/LinkedIn)
```
Slide 1: Bold headline + hook (question or shocking stat)
Slide 2-8: One key point per slide with visual
Slide 9: Summary / key takeaways
Slide 10: CTA + profile tag
```

#### Short-Form Video Script
```
[0-3s] HOOK: "Did you know..." / "Stop doing this..." / Pattern interrupt
[3-15s] CONTEXT: Set up the problem or curiosity gap
[15-45s] VALUE: Deliver the insight, tip, or story
[45-55s] PAYOFF: The resolution or key takeaway
[55-60s] CTA: "Follow for more" / "Comment below" / "Save this"
```

#### Twitter/X Thread
```
Tweet 1: Bold claim or question (hook)
Tweet 2-9: One point per tweet, numbered
Tweet 10: Summary + CTA ("Repost if useful, Follow for more")
```

## Viral Content Patterns

### What Makes Content Go Viral
1. **Emotional trigger** ‚Äî Joy, surprise, anger, inspiration
2. **Relatability** ‚Äî "This is so me" factor
3. **Shareability** ‚Äî Would someone tag a friend?
4. **Timing** ‚Äî Riding trends early (not late)
5. **Pattern interrupt** ‚Äî Unexpected visual or statement
6. **Simplicity** ‚Äî One clear message
7. **Native format** ‚Äî Looks like it belongs on the platform

### Trending Content Formats (2026)
- **AI-enhanced UGC** ‚Äî Authentic feel with polished quality
- **Interactive carousels** ‚Äî Gamified, quiz-style slides
- **Behind-the-scenes** ‚Äî Raw, unfiltered process content
- **Micro-documentaries** ‚Äî 60s storytelling reels
- **POV content** ‚Äî First-person perspective videos
- **Split-screen reactions** ‚Äî Side-by-side commentary
- **Text-heavy reels** ‚Äî Typography-focused motion content
- **Silent videos with captions** ‚Äî 85% watch without sound
